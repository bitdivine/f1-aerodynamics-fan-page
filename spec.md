# F1 Aerodynamics Fan Page

## Overview
A static educational fan page about Formula 1 car aerodynamics with informative content and visual diagrams.

## Features

### Homepage Layout
- Header with site title "F1 Aerodynamics Explained"
- Three main educational sections:
  - **Downforce**: Brief explanation of how F1 cars generate downforce for better grip
  - **Drag**: Concise overview of aerodynamic drag and its impact on speed
  - **Wings**: Simple description of front and rear wing functions
- Footer crediting the site as a fan-made educational page

### Visual Design
- Clean, readable layout with white background
- Red and black color accents inspired by F1 branding
- Each section includes illustrative diagrams or images
- Responsive design for different screen sizes

### Content
- All content in English
- Educational focus on aerodynamics concepts
- Concise, accessible explanations suitable for F1 fans
- Static content (no dynamic features required)

## Technical Requirements
- Frontend-only application (no backend needed)
- Static content with no user interaction or data persistence
- Responsive web design
